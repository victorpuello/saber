como Componentes

01
Texto base simple
Un contexto únicamente textual, sin tabla, gráfico ni layout especializado. Es el caso base del producto actual y sirve tanto para preguntas individuales como para bloques de preguntas.
LC SC LC CN QC.textOnly()
Material de lectura base
4
Enunciado
Respuesta correcta
La región Andina concentra el 72.1% de la población pero solo el 12% del territorio, lo que evidencia alta densidad poblacional en contraste con su extensión. Las demás opciones contradicen los datos de la tabla.
codeAPI — QC.textOnly(config)
paragraphs required
string[]
Párrafos de texto renderizados en orden. Cada elemento se imprime como un <p>.
instruction optional
string
Instrucción breve en cursiva sobre el material de lectura.
footer optional
string
Nota de cierre o fuente bajo el texto.
const { badge, contextHTML } = QC.textOnly({ instruction: "Lee el siguiente texto y responde la pregunta.", paragraphs: [ "Durante la última década, varias instituciones educativas rurales han reorganizado sus horarios para integrar proyectos de lectura, huertas escolares y trabajo colaborativo.", "Aunque el tiempo destinado a cada actividad es menor que en el modelo tradicional, los docentes reportan una mayor participación estudiantil y mejor transferencia entre áreas." ], footer: "Fuente: Informe institucional reelaborado con fines académicos." });
02
Texto continuo + tabla de datos
Un pasaje de lectura continuo acompañado de una tabla estadística o comparativa. El estudiante debe integrar la información del texto con los datos de la tabla para responder.
MAT LC SC CN QC.proseTable()
Distribución poblacional colombiana
7
Enunciado
Respuesta correcta
El sonido viaja 1.531 m/s en agua de mar. En 2 s recorre 3.062 m (ida+vuelta), por lo que la distancia al obstáculo es 3.062 ÷ 2 = 1.531 m.
codeAPI — QC.proseTable(config)
intro optional
string
Párrafo de contexto (qué está investigando el estudiante).
headers required
{ text, num? }[]
Encabezados de columna.
rows required
{ cells: { text, num?, hi?, bold? }[] }[]
bold:true para la columna de nombre del elemento.
source optional
string
Nota de fuente en cursiva bajo la tabla.
const { badge, contextHTML } = QC.scientificTable({ intro: "Un estudiante investiga la propagación del sonido...", headers: [ { text:"Medio" }, { text:"Estado" }, { text:"Velocidad (m/s)", num:true }, { text:"Temp. (°C)", num:true } ], rows: [ { cells:[{ text:"Vacío", bold:true }, { text:"—" }, { text:"0", num:true }, { text:"—" }] }, { cells:[{ text:"Aire", bold:true }, { text:"Gaseoso" }, { text:"343", num:true }, { text:"20" }] } // ...more rows ], source: "Fuente: Datos de laboratorio reelaborados con fines académicos." });
03
Gráfica de barras
Una gráfica de barras agrupadas construida en CSS puro (sin canvas ni librería externa). Permite comparar valores entre categorías en uno o más períodos de tiempo.
MAT SC CN QC.barChart()
Producción agrícola por departamento
9
Enunciado
Respuesta correcta
Nariño: (190−100)/100 = 90%. Boyacá: ≈69%. Antioquia: ≈46%. Aunque Antioquia creció más en términos absolutos, el mayor crecimiento porcentual es de Nariño (90%).
codeAPI — QC.barChart(config)
groups required
{ label, bars: { value, color, label? }[] }[]
Cada grupo es un clúster de barras (posición horizontal). bars contiene las barras del clúster (por ej. año 2018 y 2024).
maxVal required
number
Valor máximo del eje Y (usado para escalar la altura de barras). Usa el valor más alto de los datos + ~10%.
legend required
{ color, label }[]
Leyenda mostrada bajo las barras. Debe coincidir con el orden de bars.
height optional
number (default: 100)
Altura en px del área de barras. Aumenta si los valores necesitan más resolución visual.
const { badge, contextHTML } = QC.barChart({ title: "Producción agrícola por departamento (miles de toneladas)", maxVal: 430, groups: [ { label:"Antioquia", bars:[{ value:280, color:"#1d4ed8" },{ value:410, color:"#60a5fa" }] }, { label:"Valle", bars:[{ value:240, color:"#1d4ed8" },{ value:350, color:"#60a5fa" }] }, // ...more groups ], legend: [{ color:"#1d4ed8", label:"2018" },{ color:"#60a5fa", label:"2024" }] });
04
Aviso público institucional
Texto no continuo que replica avisos, letreros, horarios o reglamentos institucionales. Evalúa comprensión pragmática a nivel A1/A2 en Inglés (Part 1). También válido para LC (textos discontinuos).
ING · Parte 1 LC QC.publicNotice()
Institutional sign comprehension
2
Question
Correct answer
The notice shows Saturday hours (9 a.m.–1 p.m.) and states CLOSED on Sundays. Only option B correctly reflects this.
codeAPI — QC.publicNotice(config)
lang optional
'es' | 'en' (default: 'es')
Controla el badge: 'Public Notice · Part 1' en inglés o 'Aviso Público' en español.
header required
string
Título/nombre del aviso (ej. nombre de la institución). Admite HTML (emojis, strong).
rows required
{ label, value }[]
Filas de información del aviso. label: etiqueta izquierda; value: dato derecho.
note optional
string
Nota destacada en recuadro verde (ej. "Free Wi-Fi"). Admite HTML.
instruction optional
string
Instrucción en cursiva sobre el aviso (ej. "Read the following notice…").
const { badge, contextHTML } = QC.publicNotice({ lang: "en", instruction: "Read the following public notice and answer the question.", header: "🏛️ Riverside Public Library", rows: [ { label:"Monday – Friday", value:"8:00 a.m. – 6:00 p.m." }, { label:"Saturday", value:"9:00 a.m. – 1:00 p.m." }, { label:"Sunday & holidays", value:"CLOSED" } ], note: "📶 Free Wi-Fi available for all registered members", footer: "Membership is free — register at the front desk" });
05
Diálogo conversacional con burbujas
Una conversación entre dos o más personajes, renderizada con burbujas de chat estilo mensajería. Evalúa comprensión de diálogos en inglés (ING Part 3) o de textos dramáticos en LC.
ING · Parte 3 LC QC.dialogue()
Conversational English completion
14
Question
Correct answer
Marco says "My team is presenting a project about solar energy." Option A describes Ana's role (robotics), and option B is not mentioned in the dialogue.
codeAPI — QC.dialogue(config)
speakers required
{ [key]: { name, initials, bg, color } }
Mapa de hablantes. key es el identificador usado en lines[].speaker. bg/color son para el círculo avatar.
lines required
{ speaker, side: 'left'|'right', text }[]
Lista ordenada de turnos. side:'right' alinea la burbuja a la derecha (interlocutor secundario).
lang optional
'es' | 'en'
Afecta el badge: 'Dialogue · Part 3' o 'Diálogo'.
const { badge, contextHTML } = QC.dialogue({ lang: "en", instruction: "Read the conversation and answer the question.", speakers: { ana: { name:"Ana", initials:"A", bg:"#dbe1ff", color:"#004ac6" }, marco: { name:"Marco", initials:"M", bg:"#ecfdf5", color:"#047857" } }, lines: [ { speaker:"ana", side:"left", text:"Are you going to the science fair?" }, { speaker:"marco", side:"right", text:"Yes! My team is presenting solar energy." } // ...more lines ] });
06
Cloze gramatical (fill-in-the-blank)
Un texto con huecos numerados donde el estudiante elige la opción gramaticalmente correcta para cada espacio. El hueco activo (que se está respondiendo) se resalta en azul.
ING · Parte 4 QC.cloze()
Grammar in context — Past tenses
23
Question
Correct answer
"Last summer" is a specific past time marker → Simple Past. "Joined" is correct. "Has joined" (Present Perfect) requires no specific time reference. "Was joining" (Past Continuous) doesn't fit the narrative context.
codeAPI — QC.cloze(config)
segments required
({ type:'text', text } | { type:'blank', n })[]
Alterna segmentos de texto y huecos numerados. El párrafo completo se reconstruye concatenando estos segmentos.
activeBlank required
number
Número del hueco (1-based) que se está contestando actualmente. Se resalta en azul.
footer optional
string
Nota aclaratoria bajo el texto (ej. "Answer blank (1) only — the correct form of join").
const { badge, contextHTML } = QC.cloze({ lang: "en", instruction: "Read the text. Choose the best option for each blank.", activeBlank: 1, segments: [ { type:"text", text:"Last summer, Maria " }, { type:"blank", n:1 }, { type:"text", text:" a volunteer program. Every morning, she " }, { type:"blank", n:2 }, { type:"text", text:" to the beach." } ], footer: "Answer blank (1) only — the correct form of the verb <em>join</em>." });
07
Figura geométrica SVG
Una figura vectorial (triángulo, círculo, gráfico de fuerza, diagrama de circuito, etc.) incrustada como SVG inline. Permite mostrar medidas etiquetadas, ángulos y relaciones geométricas directamente en el contexto.
MAT — Geometría CN — Física/Diagramas QC.geometry()
Triángulo rectángulo — Razones trigonométricas
16
Enunciado
Respuesta correcta
Teorema de Pitágoras: AC² = AB² + BC² = 144 + 25 = 169. Por lo tanto AC = √169 = 13 cm. Las opciones B, C y D aplican operaciones incorrectas sobre los catetos.
codeAPI — QC.geometry(config)
svgContent required
string (raw SVG markup)
El tag <svg> completo incluyendo viewBox, width y height. Usa font-family="Inter" para las etiquetas de texto.
intro optional
string
Instrucción antes de la figura (ej. "Observe el siguiente triángulo y responda…").
caption optional
string
Nota descriptiva debajo de la figura (ej. "El ángulo recto está en B. AB = 12 cm, BC = 5 cm").
const { badge, contextHTML } = QC.geometry({ intro: "Observe el siguiente triángulo rectángulo y responda.", svgContent: ` <svg width="320" height="220" viewBox="0 0 320 220"> <polygon points="40,180 280,180 280,40" fill="#eff6ff" stroke="#1d4ed8" stroke-width="2.5"/> <rect x="263" y="163" width="17" height="17" fill="none" stroke="#1d4ed8" stroke-width="2"/> <text x="145" y="200" font-family="Inter" font-size="13" fill="#505f76">AB = 12 cm</text> <text x="287" y="118" font-family="Inter" font-size="13" fill="#505f76">BC = 5 cm</text> <text x="130" y="100" font-family="Inter" font-size="13" fill="#004ac6" font-weight="600" transform="rotate(-22,130,100)">AC = ?</text> </svg>`, caption: "El ángulo recto está en B. Catetos: AB = 12 cm, BC = 5 cm." });
08
Tabla científica
Datos experimentales o de referencia presentados en una tabla estructurada con contexto científico. El estudiante usa los datos para calcular, comparar o inferir propiedades.
CN MAT QC.scientificTable()
Velocidad del sonido en distintos medios
19
Enunciado
Respuesta correcta
El sonido viaja 1.531 m/s en agua de mar. En 2 segundos recorre el doble de la distancia al obstáculo, por eso la distancia correcta es 1.531 m.

{ text, num? }[]

(°C)", num:true } ], rows: [ { cells:[{ text:"Aire", bold:true }, { text:"Gaseoso" }, { text:"343", num:true }, { text:"20", num:true }] }, { cells:[{ text:"Agua de mar", bold:true }, { text:"Líquido" }, { text:"1.531", num:true }, { text:"20", num:true }] } ], source: "Fuente: Datos de laboratorio reelaborados con fines académicos." });
