# Simulador Saber 11 — UI Kit

Interactive click-through prototype of the Saber 11 Simulador frontend.

## Screens Included

| Screen | Description |
|---|---|
| Login | Two-column layout with scholar-gradient panel + form |
| Student Dashboard | Sidebar nav, hero card, metrics grid, area performance |
| Diagnostic Session | Two-pane exam interface (reading context + question + options) |
| Admin Dashboard | Institution metrics hero, question bank table |

## Usage

Open `index.html` to start from the Login screen and navigate through all screens.

## Tech Notes

- Pure HTML/CSS/JS (no build step)
- Material Symbols Outlined icon font from Google CDN
- Inter font from Google CDN
- Tailwind-like utilities implemented as inline styles matching the actual codebase tokens
- All colors from `colors_and_type.css`
